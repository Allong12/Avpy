=================
 Pyav Examples
=================


* mediaInfo

Retrieve media infos and libav infos

* outputPgm

Decode first video stream of media
Write pgm images

* outputWav

Decode first audio stream of media
Write wav file

* alsaPlay

Decode first audio stream of media
Play it with alsa (Linux only)

* outputPygame

Decode first video stream of media
outputPygame.py -> write jpg images
outputPygame2.py -> dump video player
